# kitameraki_recruitment_bagasPrabowo

## Prerequisites
- NPM & NodeJs
- ExpressJS
- MongoDB

## How To Run
1. Copy the .env.example to .env and fill the .env file
```
cp .env.example .env
```
2. instal the dependency
```
npm i
```
3. run the project
- Development
```
npm run dev
```
- Production 
```
npm start
```
